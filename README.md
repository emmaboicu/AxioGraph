AxioGraph


AxioGraph este o aplicație web educațională pentru realizarea graficelor pe hârtie milimetrică digitală.

Aplicația permite:
- alegerea scalei pe axele OX și OY;
- afișarea automată a reperelor pe scară și verificarea corectitudinii scalei alese;
- marcarea punctelor experimentale;
- trasarea unei drepte de tendință;
- alegerea punctelor pentru determinarea pantei;
- marcarea intersecțiilor cu axele;
- trasarea unei curbe de tendință;
- salvarea și încărcarea lucrării în format `.axio`.

Punctele desenate de aplicație fac snap la grilă, așa cum elevul ar trebui să procedeze și în reprezentarea manuală pe hârtie milimetrică.


Aplicația este destinată învățării modului corect de reprezentare grafică a datelor experimentale.

Elevul exersează:
- alegerea unei scări potrivite pe axele OX și OY;
- înțelegerea rezoluției graficului;
- observarea erorilor introduse de o scară prea grosieră;
- reprezentarea punctelor experimentale pe grilă;
- observarea tendinței punctelor experimentale;
- trasarea unei drepte de tendință sau a unei curbe de tendință;
- alegerea unor puncte potrivite pentru determinarea pantei;
- determinarea intersecțiilor dreptei cu axele.

Aplicații

Am folosit AxioGraph la lucrări experimentale (virtuale):

- Determinarea constantei elastice a unui resort prin metoda micilor oscilații (din graficul T²(m));
- Determinarea constantei elastice a unui resort prin metoda statică.
- Determinarea tensiunii electromotoare E și a rezistenței interne r pentru o sursă din graficul U (I);
- Studiul si determinarea rezistenței unui bec neohmic;
- Determinarea constantei lui Planck folosind caracteristicile curent-tensiune ale unei celule fotoelectrice pentru două radiații   monocromatice diferite.


Utilizarea aplicației

Deschide aplicația din linkul GitHub Pages al proiectului.

Pentru continuarea lucrului, folosește:
- Save pentru salvarea lucrării;
- Load pentru încărcarea fișierului `.axio` salvat anterior.
